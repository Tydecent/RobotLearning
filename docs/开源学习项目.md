
## 四、开源学习项目

以下开源项目可用于赛前自学与实训，按赛题模块分类。

宇树官方文档：

[宇树科技 G1 开发者概览](https://support.unitree.com/home/zh/G1_developer/overview)

宇树 SDK 仓库：

[Unitree Robotics GitHub](https://github.com/unitreerobotics?q=&type=public&language=&sort=)

### 4.1 与赛题模块对应


| 模块  | 核心能力                            | 推荐学习方向                              |
| --- | ------------------------------- | ----------------------------------- |
| A   | 修改 URDF → 编译 → Gazebo 加载 → 截图验证 | 官方 URDF 教程、Gazebo 集成教程、G1 模型        |
| B   | OCR 图片关键信息提取、Python 图像处理、调用机器人动作接口 | PaddleOCR、EasyOCR、OpenCV、Unitree G1 ROS2 示例；竞赛平台无开源替代 |
| C   | 加载场景、调 Nav2 参数、Rviz2 观察避障       | Nav2 官方示例、TurtleBot3 仿真             |

#### 模块 B：脚本与动作控制


| 项目          | 链接                                                                                                                                                                                                                                                                                              | 说明                            |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| PaddleOCR（高优先级） | [GitHub：PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR) | 中文 OCR 与文档解析能力强，适合练习“图片操作说明 → 文字 → 关键信息提取” |
| EasyOCR（高优先级） | [GitHub：EasyOCR](https://github.com/JaidedAI/EasyOCR) | Python 上手简单，支持简体中文和英文，适合快速跑通样题任务一的 OCR 流程 |

| OpenCV      | [GitHub：OpenCV](https://github.com/opencv/opencv) · [中文：OpenCV-Python 入门（掘金）](https://juejin.cn/post/6844903512007143432)                                                                                                                      | 图片预处理、裁剪、二值化、降噪，可配合 OCR 提高识别效果 |

**说明**：模块 B 任务一优先练习 OCR 图片文字识别与关键信息抽取，建议先用 EasyOCR 快速跑通流程，再用 PaddleOCR 提升中文识别与文档解析能力。中慧云启竞赛平台 **10** 为赛项定制软件，无公开开源替代；模块 B 最终以赛方平台与接口文档为准。
